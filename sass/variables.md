# Variables

```css
$font-stack: Helvetica, sans-serif;
$primary-color: #333;

body {
    font: 100% $font-stack;
    color: $primary-color;
}

Can also:
use "Black"
Units "1.0em"
$d: Haelvetica, sans-serif;
$e: 1px #000 Solid 0 0; // For border

Also can use operations like +, -, *, /
$font-size: 4px + 4
```
